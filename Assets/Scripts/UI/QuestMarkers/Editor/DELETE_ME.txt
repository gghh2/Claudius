# Fichier à supprimer

Ce fichier `QuestMarkerSystemCleaner.cs` fait référence à l'ancien système et n'est plus nécessaire.
Il doit être supprimé pour éviter les erreurs de compilation.

## Pour supprimer ce fichier :

1. Dans Unity, allez dans : `Assets/Scripts/UI/QuestMarkers/Editor/`
2. Supprimez `QuestMarkerSystemCleaner.cs` et son `.meta`

## Alternative : Utiliser QuestMarkerCleaner.cs

Le fichier `QuestMarkerCleaner.cs` dans le même dossier est le bon outil de nettoyage à utiliser.
Il permet de supprimer tous les anciens fichiers obsolètes.

Utilisez : `Tools > Quest System > Clean Quest Marker Files`
